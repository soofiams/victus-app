# Victus App – Flutter + PHP API

Projeto desenvolvido como aplicação mobile/web com Flutter, ligado a uma API em PHP e base de dados SQL.

## Funcionalidades
- Login de utilizador
- Dashboard com eventos
- Biblioteca de vídeos
- Player de vídeo (subpágina da biblioteca)
- Adição de eventos com botão "+"
- Interface baseada em Figma (cores e estética claras/rosa)

## Tecnologias utilizadas
- Flutter (Frontend)
- PHP (API)
- MySQL / SQL
- XAMPP (servidor local)
- GitHub (controlo de versão)

## Estrutura do Projeto
/api → Backend (API em PHP, controllers, ligação à base de dados)  
/api/app/flutter_application_1 → Aplicação Flutter (Frontend)
